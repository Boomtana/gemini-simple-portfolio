# Simple Portfolio Website

A responsive personal portfolio website for a Mathematics teacher, featuring a modern dark theme with light mode toggle and project showcases.

## Features
- **Modern Dark Theme**: Default dark theme with high-contrast elements.
- **Theme Toggle**: Switch between dark and light modes with a UI toggle.
- **Responsive Layout**: Designed to work on various screen sizes.
- **Projects Section**: Showcase of teaching-related projects.
- **Contact Details**: Easy access to contact information.

## Technologies Used
- HTML5
- Vanilla CSS
- Vanilla JavaScript
- JetBrains Mono Font

## Setup
1. Clone the repository: `git clone https://github.com/Boomtana/gemini-simple-portfolio.git`
2. Open `index.html` in your web browser.
